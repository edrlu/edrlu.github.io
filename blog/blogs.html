Cross Entropy: A Brief Mathematical Breakdown

Mar 15, 2025
8 min read

Cross entropy is a fundamental concept in information theory and machine learning, serving as both a measure of information content and a loss function for classification tasks. Understanding its mathematical foundations provides crucial insights into why it works so effectively in training neural networks.

Information Theory Foundations

Cross entropy originates from Claude Shannon's information theory. The entropy of a probability distribution P is defined as:

H(P) = -∑ P(x) log P(x)

This measures the average information content or "surprise" in a distribution. Cross entropy extends this concept to measure the difference between two probability distributions P and Q:

H(P,Q) = -∑ P(x) log Q(x)

Machine Learning Applications

In classification tasks, cross entropy loss compares the predicted probability distribution Q (model output) with the true distribution P (one-hot encoded labels). For a single training example with true class i:

L = -log(Q(y_i))

Where Q(y_i) is the predicted probability for the correct class. This formulation has several advantages:

Gradient Properties: The cross entropy loss provides strong gradients when predictions are wrong and smaller gradients when predictions are correct, leading to efficient learning.

Probabilistic Interpretation: The loss directly corresponds to the negative log-likelihood, making it a natural choice for probabilistic models.

Convexity: For linear models, cross entropy loss is convex, guaranteeing a global minimum.

Relationship to KL Divergence

Cross entropy is closely related to Kullback-Leibler (KL) divergence:

D_KL(P||Q) = H(P,Q) - H(P)

Since H(P) is constant for fixed true labels, minimizing cross entropy is equivalent to minimizing KL divergence, which explains why cross entropy effectively measures distributional similarity.

This mathematical framework makes cross entropy an indispensable tool in modern machine learning, providing both theoretical justification and practical effectiveness in training classification models.
