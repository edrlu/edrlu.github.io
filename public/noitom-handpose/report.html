Hand-Pose Annotation - Internal Pipeline Report

Egocentric clip 695026376e - 1920x1440 - 30s window (frames 350-1249) - 4 pipelines compared

We built an internal hand-pose annotation pipeline to reduce dependence on paid SomaStacks annotation. This report compares four pipelines on the same clip across coverage, ground-truth accuracy, and temporal stability, and states where each one wins.

The dashboard provides the visual check: the same 30-second window, synchronized across all four rendered overlays.

Part One - Coverage

Share of frames a hand is tracked; higher is not the same as better landmarks.

The hybrid finds the most hands; MANO and SomaStacks are more conservative.

Part Two - Accuracy vs Ground Truth

The Noitom headset records its own on-device 26-joint hand tracking. We project it into the image, calibrated to ~23px, and measure our 2D landmark error: a real accuracy metric, unlike coverage. SomaStacks clips have no independent GT, so this is measured on the Noitom session.

MediaPipe 30.1px 52%

On easy near-field data all internal pipelines are ~28-30px, so they are comparable; the mesh model does not help much there.

Part Three - Temporal Stability

Jitter is on-screen high-frequency noise in pixels around the smoothed trajectory; lower means steadier.

ModelJitter px
MediaPipe 2.1px
WiLoR-hybrid 1.8px
WiLoR-MANO 3.4px
SomaStacks 3.1px

Low jitter can be misleading: MediaPipe and hybrid look steady partly because they smooth over misplaced landmarks.

The endorsed WiLoR-MANO was originally ~17-20px jitter; after root-relative plus stronger finger smoothing it is now 3.4px, matching SomaStacks at 3.1px: clean and steady.

Part Four - Landmark Quality

On hard motion-blur frames, WiLoR-MANO draws clean anatomical skeletons on par with SomaStacks, while the hybrid catches more hands but draws them roughly and MediaPipe misses most. This is the visual axis; use the dashboard for the frame-by-frame comparison.

Methodology - The Endorsed Pipeline

WiLoR YOLO detector-> WiLoR ViT-> MANO mesh-> temporal smoothing-> HDF5 export-> QC overlay

HDF5 schema matching the SomaStacks contract:

  • camera/intrinsic
  • camera/extrinsic_w2c
  • transforms/camera
  • mediapipe21/{joints,is_right,visibility,confidence}
  • pose6d/{wrist_position,wrist_rotation_matrix,wrist_quaternion}
  • mano/{global_orient,hand_pose,betas,transl,is_right,visibility}
  • meta/{frame_indices,timestamps}

Verdict / Recommendation

No single model wins every axis: hybrid has the best coverage, while WiLoR-MANO has the best landmark quality and now matches SomaStacks stability. Recommended next step: feed WiLoR-MANO's reconstruction with the hybrid's more aggressive detector to lift coverage while keeping quality; optional temporal motion prior (HMP) for further stability.
